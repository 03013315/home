
  
# 软件使用手册

“工欲善其事，必先利其器”,软件开发少不了软件工具。首先，需要会用，不然，纸上谈兵是和软件无缘的。   进一步是擅用软件工具，提高学习和工作的效率。 
     
课程软件使用手册是同学们会用、擅用软件的小帮手，助力同学们“动”起来！
     
以后的学习工作中，使用那些工具好？ 那些工具能助你更好的开展工作就是答案。不要做工具的奴隶。
